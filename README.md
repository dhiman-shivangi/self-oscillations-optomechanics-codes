This repository contains a set of Mathematica notebooks that have been used to obtain results for the paper "Self-sustained Oscillations of a Non-linear Optomechanical System in the Low Excitation Regime". The codes are written in 'Wolfram Mathematica 13.1' and has been tested on 'Wolfram Mathematica 14.1' as well.

# Reading the notebook
To run the files, you need Wolfram Mathematica (https://www.wolfram.com/mathematica/) on your desktop. Alternatively, the files can be read using the free Wolfram Player (https://www.wolfram.com/player/). The installation instructions for both are available at the respective webpages. We have also provided the files in pdf format to ensure accessibility.

# Usage
To use the code, simply open the .nb notebooks and evaluate the cells sequentially to obtain the results. The expected output for each of the codes should already be visible as soon as you open the files. There are two codes provided -
1. Sample code-phase diagram- Fig2 : This takes about 5 minutes to compile on a 11 core device.
2. Sample code-simulation-Fig4, IIIb : This code can take upto a day to compile for the given parameters. If you change the parameters, the compilation time can vary significantly.

You can change the parameters in the code for your setups and rerun to obtain the results for your case. The current parameters in the code can be used to reproduce Fig 2 and Figure 4, IIIb in our manuscript.
